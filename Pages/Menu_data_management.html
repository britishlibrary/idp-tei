Extract and save TEI files

Script to extract TEI files from 4D database blobs and place copies in the idp-tei repository: http://localhost/D_DRIVE_BRITISH_LIBRARY/bl github group/bl_github_clones/idp-tei/Scripts/PHP/extract_and_save_tei_xml.php
