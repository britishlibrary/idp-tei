Incipit ( 1r.1 ): //__//'phags pa spyan ras gzigs dbang phyug la bstod pa'
This short prayer ( stotra ) to Avalokiteśvara does not contain any elements which would indicate a classification as Vajrayāna . Nevertheless, it is written in the same hand, on the same size paper, as the Avalokiteśvara dhāraṇī text in ITJ_214 . This prayer is not similar to any of the Avalokiteśvara stotra texts in the bstan 'gyur .
ITJ_214 is written in the same hand, on the same size folio. However, there does not appear to be a direct relationship with this manuscript.
