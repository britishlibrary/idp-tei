de bzhin gshegs pa'I gtsug tor gdugs dkar po
['phags pa de bzhin gshegs pa'i gtsug tor nas byung ba'i gdugs dkar mo can gzhan gyis mi thub ma zhes bya ba'i gzungs]
[Ārya-tathāgatoṣṇīṣasitātapatre-aparājitā-nāma-dhāraṇī]
Incipit ( Rf.1r.1 ): kha sa me kha sa me/
This manuscript comprises only three folios of a concertina. Six further folios from the same concertina appear in the Pelliot collection ( PT_57 ). The Pelliot folios come before this one, with a lacuna of several folios between them. The first text on the recto is the present item, which is similar to other versions in the Dunhuang collections (see ITJ_323/1 for a brief discussion of this text). Only the closing lines and the title appear in this manuscript.

[phags pa mdangs phyir 'phrog pa] zhes bya ba'i mdo
[Ārya-ojapratyaṅgiranisūtra]
Incipit ( Vf.1r.1 ): zhes bya ba'i mdo /
This dhāraṇī sutra is missing the first part of the title, but can be identified with the canonical version ( P_295 ), as it is almost identical to it. It is incomplete at the end, though only a few lines are missing (see P_295: 62a.1 ). The text that follows this in the further folios from this concertina found in the Pelliot collection is the Candanāṅga dhāraṇī (see ITJ_337/1 )
